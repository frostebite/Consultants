# Consultants

Claude Code consultant agent definitions for the Frostebite studio.

## Categories

- **engineering/** - Engineering consultants (performance, architecture, infrastructure)
- **fans/** - Fan community and player perspective consultants
- **player/** - Player psychology and experience consultants
- **porting/** - Platform porting and compatibility consultants
- **subject-matter-experts/** - Subject matter experts (military, art, fantasy, film, systems)
- **systems/** - Systems design and governance consultants

## Usage

This repository is consumed as a Git submodule at `.claude/consultants` in the GameClient monorepo.
